# Jira Alternative

> The simpler, faster alternative to Jira

A focused alternative to Jira that fixes the top user complaints. Built for Project Management who need the core functionality without the bloat.

## Features

- **Core Workspace** — The main product area — fast, clean, no bloat.
- **Onboarding Flow** — Get users to first value in under 5 minutes.
- **Simple Billing** — Transparent flat-rate pricing via Stripe.
- **Team Sharing** — Share and collaborate without per-seat fees.
- **Integrations** — Connect to the tools your users already use.

## Tech Stack

- **Frontend:** React + Vite + TypeScript + Tailwind CSS
- **Backend:** FastAPI (Python 3.12)
- **Database:** PostgreSQL
- **Auth:** Supabase Auth
- **Payments:** Stripe
- **Deployment:** Docker + Railway

## Getting Started

```bash
cp .env.example .env
docker compose up --build
```

## Target Audience

Indie developers and small teams frustrated with Jira's complexity and pricing.

## Pricing

Free tier (limited). Pro at $9/month flat. No per-seat fees.
